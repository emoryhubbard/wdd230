Typography

The major considerations for choosing type for the White Oak Chamber site were finding fonts that were not overly common, yet readable, modern, and non-distracting to visitors. Two were selected, one of which will be used for headings, and another for paragraph-level information, as well as other related needs. Both fonts were chosen from google's font API system, accesible from the following site:

fonts.google.com

ABeeZee Font

The ABeeZee Font was chosen to be used for the headings, accents, and other areas of emphasis on the site. It has a unique character that can help the site remain distinctive, yet is not distracting to the viewer. To use it, include the following import statement, and the following name for the font-family when you need it:

@import url('https://fonts.googleapis.com/css2?family=ABeeZee&family=Mukta+Malar&display=swap');

font-family: 'ABeeZee', sans-serif;

Mukta Malar Font

The Mukta Malar Font was chosen to be used for the paragraphs, some accents, and other paragraph-level content on the site. Whenever there are larger lines or blocks of text this font will generally be used for readability. It is only slightly distinctive. To use it, include the following import statement, and the following name for the font-family when you need it:

@import url('https://fonts.googleapis.com/css2?family=ABeeZee&family=Mukta+Malar&display=swap');

font-family: 'Mukta Malar', sans-serif;
